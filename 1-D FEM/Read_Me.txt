The file "FEM.mlx" is the main finite element function which is to be called to solve the problem. It takes 2 inputs, the input file and the output file.
First enter the inputs in the "input_file.mlx" file and run the file. Then run the FEM function with the input file name as created by you in the "input_file.mlx" while saving the workspace variables.
"FEM_runner.mlx" shows one example to run the code.
